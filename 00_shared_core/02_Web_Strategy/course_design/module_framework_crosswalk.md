# Web Strategy Module Framework Crosswalk

## Purpose

This crosswalk helps instructors create framework contracts for Web Strategy discussions, Brand Pods, website/app operator activities, case-study applications, and review activities. It does not change the syllabus, create assignments, alter point values, change due dates, or add rubric categories.

The Web Strategy syllabus remains authoritative for module sequence, assigned chapters, assessments, due dates, and workload. Use this crosswalk with the Overclocked framework library / appendix, chapter overview, canonical team assessment rubric, digital operating-role architecture, and brand/surface/scenario design guide.

## Framework Contract Guidance

Each graded activity should identify one required framework, any role-specific framework emphasis, optional extension frameworks, and frameworks not to force. If students choose a framework, provide a bounded set. Role lenses, brands, and digital surfaces do not replace framework application.
