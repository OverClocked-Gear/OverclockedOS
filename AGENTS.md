# Overclocked CourseOS Instructions

This repository supports two related courses:
- Web Strategy: undergraduate, 15-week, asynchronous
- Digital Strategy: graduate, 8-week, synchronous

Use Overclocked as the primary conceptual spine.

Do not recreate Canvas module folders unless explicitly requested.

Use the syllabi and course_crosswalk.yaml files as the authoritative course maps.

Separate:
- shared core concepts and frameworks
- industry intelligence
- Web Strategy course overlay
- Digital Strategy course overlay
- assessment-secure materials

When generating content:
1. Connect outputs to Overclocked frameworks.
2. Map outputs to Web Strategy modules and Digital Strategy sessions.
3. Add a website/app operator lens when useful.
4. Keep undergraduate async workload light.
5. Make graduate synchronous outputs more workshop- and decision-oriented.
6. Mark proprietary reports and data as instructor-only unless cleared.
7. Do not invent citations, readings, dates, source facts, point values, or policies.
8. Put quiz items, answer keys, and assessment seeds in assessment-secure.
